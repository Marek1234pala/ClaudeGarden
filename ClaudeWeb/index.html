You: Write a Python function that fetches a URL with retry logic and exponential back-off.
Claude: 
import time
import requests

def fetch_with_retry(url, max_retries=5, base_delay=1.0):
    for attempt in range(max_retries):
        try:
            response = requests.get(url, timeout=10)
            response.raise_for_status()
            return response
        except requests.RequestException as e:
            if attempt == max_retries - 1:
                raise
            delay = base_delay * (2 ** attempt)
            print(f"Attempt {attempt + 1} failed: {e}. Retrying in {delay}s…")
            time.sleep(delay)

This doubles the wait time after each failure (1 s → 2 s → 4 s …), preventing thundering-herd issues. Raises the original exception if all retries are exhausted.

You: I have two job offers. One pays $120k with equity, the other $145k with no equity. How should I think about this?
Claude:

The $25k salary gap is real money — roughly $2,083/month after tax. Here is how to weigh the equity side:

  • Stage matters: Seed-stage equity can be worth 10–100× or zero. Series B+ is more predictable but less upside.
  • Vesting schedule: 4-year / 1-year cliff is standard. Leaving before the cliff = nothing.
  • Dilution: Ask what % the option pool is, and how many funding rounds are planned.
  • Liquidity horizon: Private company equity may be illiquid for 5–10 years.

Rule of thumb: if the equity grant's expected value at exit doesn't cover at least 2–3 years of the salary gap, the higher cash offer is likely the safer choice — unless the mission or role learning matters more to you.
